# Contributing to tsp-types

Thank you for your interest in contributing to **tsp-types**!

This repository provides Rust type definitions for the
[Type Server Protocol (TSP)](https://github.com/nicolo-ribaudo/tc39-proposal-type-server-protocol),
generated from the canonical TypeScript protocol definition.

## Ways to Contribute

You can contribute by:

- Updating the TypeScript protocol definition
- Improving the code generators
- Fixing bugs in the generated Rust types
- Improving documentation

## Pull Request Guidelines

- Keep pull requests focused and scoped
- Avoid combining unrelated changes
- Include documentation updates where appropriate
- Be respectful of differing implementation approaches

## Code of Conduct

This project follows the
[Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/).
By participating, you are expected to uphold this code.
